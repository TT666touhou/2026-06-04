# Agent Deck — Multi-Agent Terminal 監控儀表板 安裝指南
# 正確來源：github.com/asheshgoplani/agent-deck

## 功能
Agent Deck 是一個 TUI (終端使用者介面) 工具，提供：
- 集中式儀表板，監控所有活躍的 AI Agent sessions
- 單鍵切換 Agent sessions
- Fork Claude/Gemini 對話（嘗試不同方案而不丟失上下文）
- 原生支援 Git Worktree（多分支平行開發）
- MCP 管理介面

## 安裝指令 (Linux/macOS)
```bash
curl -fsSL https://raw.githubusercontent.com/asheshgoplani/agent-deck/main/install.sh | bash
```

## Windows 安裝
Agent Deck 目前主要支援 Linux/macOS。
Windows 用戶建議使用 WSL2 (Windows Subsystem for Linux) 安裝：
1. 開啟 WSL2 終端
2. 執行上方的 curl 安裝指令
3. 在 WSL2 環境中使用 Agent Deck 管理 Agents
