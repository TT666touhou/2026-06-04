## 🎮 1. 遊戲核心概念

### 1.1 核心定位 [DRAFT]
> 一款 **2D 橫向戰術回合制遊戲**，單人體驗。
> 每回合玩家規劃一個行動（射針、跳躍、盪繩），確認後動態播放，播放結束再凍結進入下一回合。
> 玩家持有**固定數量的鋼針**，跨回合將針佈置於場景中，組合產生複雜的移動與戰鬥效果。
> **佈局策略與時機判斷**是取勝核心，而非反應速度。
> 靈感融合：**Ronin**（每回合一個行動，動態播放後凍結）× **Dimension W**（鋼針投擲精準戰鬥）× **Titan Souls**（一擊致命 + 武器回收）

**設計靈感來源：**
- **Dimension W（漫畫）** — 主角 Kyouma 的戰鬥風格：非科技、純機械、近距投擲、精準打點
- **Titan Souls** — 一擊致命、武器（針）必須回收、Boss 弱點精確打擊

### 1.2 遊戲類型 [DRAFT]
- **主類型**：2D 橫向戰術回合制（Turn-based Tactical Platformer）
- **次類型**：資源管理動作（Resource Management Action）
- **人數**：單人（已確認，MMP-003 解除）
- **維度**：2D（橫向捲軸）

### 1.3 核心設計支柱 [DRAFT]
1. **資源即工具**：鋼針是移動工具也是武器，永遠有限，迫使玩家思考
2. **多功能單一機制**：同一根針根據使用方式產生完全不同的效果
3. **戰術精確**：在慢動作中從容瞄準——挑戰來自「思考正確方案」而非「反應速度」
4. **回收循環**：使用後必須回收針（或找到補充），製造節奏張力
5. **多針交響**：多根針同時在空中產生複合效果，佈局能力決定深度上限

### 2.2 針的使用模式（回合制，2026-06-22 確認）[DRAFT]

**每回合只能射出一根針（左鍵或右鍵擇一）。**

| 用途 | 輸入 | 回合制行為 | 狀態 |
|------|------|-----------|------|
| **攻擊針** | **左鍵**拖曳瞄準 → 確認 | 1 回合：針飛行並命中目標，插入地形/敵人留在場景中 | [DRAFT] |
| **繩針（鉤抓/盪繩）** | **右鍵**點擊瞄準 → 即時固定 | 0 飛行時間：右鍵 raycast 480px 內偵測牆面，命中即時建立錨點；未命中靜默失敗（不消耗回合）| [DRAFT] |
| ~~**鋼索平台**~~ | 已移除（GAP-041） | — | 已移除 |
| **拉動物體** | TBD | MVP 排除，後期實作 | [DRAFT] |
| **地形彈射** | 針飛行命中特定地形後 | 自動彈射到新方向（反射角），繼續飛行 | [DRAFT] 保留 |

> **射出機制（回合制版）**：
> - 凍結狀態下，玩家左鍵拖曳 = 預覽攻擊針彈道（虛線 + 落點圓）
> - 玩家右鍵拖曳 = 預覽繩針彈道（虛線 + 落點圓）
> - 確認（放開 or 二次點擊）= 凍結解除，動態播放，播放結束再凍結
> - 同一回合**不能**同時確認攻擊針和繩針

### 2.3 細線（Wire）系統 [DRAFT]
**已確認（GAP-041 大幅簡化 + GAP-053 全面重寫）：**
- **只保留盪繩抓鉤**；平台機制、S 鍵穿透、Q/E/F 全部移除（GAP-041）。
- **視覺改為直線 Line2D**（GAP-053 移除 Verlet 12 段繩，改為 2 點直線）。

**帶線針物理 — 即時勾爪 + 手動收縮 + 自然鐘擺（GAP-053/064 重寫）[DRAFT]**

- **右鍵即時固定**（GAP-064）：右鍵點擊 → raycast 從玩家朝滑鼠方向最遠 480px；命中牆面 → 立即建立 WireConstraint 錨點，無針飛行時間；未命中 → 靜默失敗。
- **無自動收繩**（GAP-064）：繩長不會自動縮短，固定在命中瞬間距離。
- **手動收縮**（GAP-064）：錨點旁顯示「⬆ 收縮」UI 按鈕；點擊 → 下一回合開始時縮短 120px（可再次點擊取消）。
- **Pre/Post 約束分離**（GAP-053 核心）：
  - `_apply_wire_pre`：`move_and_slide()` 前消除朝外徑向速度 + `snap_factor = 0.35` 彈力反饋
  - `_apply_wire_post`：`move_and_slide()` 後直接以 `global_position` 硬夾位置到繩長圓弧
  - **不注入任何速度脈衝**（消除速度分段問題）
- **Player 彈弓移動 → 自動斷線**（GAP-064）：玩家選擇彈弓移動時，繩索自動釋放；攻擊針不斷線。
- **直線繩視覺**：2 點 Line2D（玩家→錨點），張力比例決定顏色與寬度（GAP-053）。
- **初始繩長**：命中瞬間精確距離（`wire_slack=0`）。
- **可調參數**：`rope_min_length=24`、`rope_snap_factor=0.35`、`REEL_STEP=120`。

## 🏃 3. 移動系統

### 3.1 彈弓移動（Slingshot）— 主要移動方式（2026-06-22 確認）[DRAFT]

> **完全取代** A/D 跑步 + Space 跳躍。

**輸入流程：**
1. 回合凍結狀態下，玩家在玩家身上按住 → 向外拖曳（預覽施力方向和強度）
2. 拖曳距離 = 力道大小（有最大值上限）
3. 放開 → 玩家被彈射出去，進入飛行動態播放
4. 飛行途中遊戲暫停（可在此時使用繩索改變方向，見 §3.2）
5. 落地 → 動態播放結束 → 再次凍結

**拖曳預覽 — 虛線路徑：**
- 拖曳過程中即時顯示**虛線路徑**，預覽玩家飛行或滑行的軌跡
- 虛線計算考慮**玩家碰撞體積**：若路徑會接觸地面，自動改為直線（滑行模式）
- 路徑接觸地面 = 變直線 → 玩家沿地面滑行
- 路徑在空中 = 保持弧線 → 玩家飛起
- 角度和拖曳力道由玩家自由控制，移動類型（滑行/飛行）由物理自動判定

**路徑特性：**
- 受重力影響的物理計算（非強制拋物線，依角度自然）
- **最大彈射射程**：約 **640px**（半個畫面寬）
- 弧線可跨多個 0.3s 回合完成，玩家在空中時每 0.3s 凍結一次，可選擇是否射繩針改向

### 3.2 繩索輔助移動（2026-06-22 確認）[DRAFT]

> 繩索有**兩種**移動用法，均為回合制動作：

**用法 A：飛行中改變路線**
- 玩家用彈弓射出後，飛行途中遊戲暫停
- 此時可射出繩針勾住牆壁錨點 → 玩家的飛行弧線被「拉向」錨點方向改變落點
- 繩針飛行動態播放完後再凍結，玩家可選擇下一步

**用法 B：獨立盪繩**
- 射出繩針命中錨點（1 回合）
- 下一回合選擇「盪繩」→ 玩家以錨點為圓心擺盪一段弧度後落地（1 回合播放）
- 可在盪繩途中再使用彈弓或另一根繩針

### 3.3 移除的移動機制
- ~~A/D 水平跑步~~：已移除，由彈弓水平彈射取代
- ~~Space 跳躍~~：已移除，由彈弓垂直彈射取代
- ~~Coyote Time / Jump Buffer~~：彈弓模式下不需要這些緩衝機制

### 5.3 訓練木人（Training Dummy）[DRAFT]

> **用途**：機制驗證用的 MVP 敵人替身，用於測試針的命中、拉動與物理手感。

**基本規格：**

| 屬性 | 值 |
|------|-----|
| 節點類型 | `CharacterBody2D` |
| 佔位尺寸 | **32 × 64 px**（棕橘色 ColorRect，與 Player 等高） |
| 質量行為 | 有重力（落地後靜止；被拉動時移動） |
| 碰撞層 | Layer 1（與牆/地板/玩家一致） |
| HP | 無（純物理測試用） |

**針的互動行為：**

| 針種類 | 命中後效果 |
|--------|-----------|
| 攻擊針（左鍵） | **插入** + **一次性擊退衝量**（`knockback_force = 500 px/s`，方向：遠離玩家；地面上額外給一個小跳起） |
| 帶線針（右鍵） | **插入**（不產生拉動；帶線針保持盪繩錨點語義） |

**擊退邏輯（2026-06-21 實作）：**
- `on_needle_embedded(n_type)` — 由 NeedleManager 呼叫；攻擊針命中時立即計算擊退方向（dummy → player 反方向），疊加 `velocity += dir × knockback_force`；在地面時額外 `velocity.y -= knockback_force × 0.35`（小跳）
- 地面摩擦：`ground_friction = 800 px/s²`，使木人自然減速停止
- 無持續效果，無需 `on_needle_removed`

**場景位置（MVP_Test.tscn）：** `Vector2(900, 704)` — 地面上、玩家右側，需要用鉤爪接近。

## 9. MVP 原型規格（最小可遊玩版本）

> MVP 目標：用純幾何形狀（ColorRect / CircleShape2D）驗證整體遊玩手感。
> 所有數值為初始設計值，進入 Godot 後可快速調整。

### 9.1 平台與操控（回合制版，2026-06-22 確認）[DRAFT]

> ⚠️ 以下為**設計目標**，與現有程式碼部分不一致，需要後續 Architect 重新規劃。

| 項目 | 規格 |
|------|------|
| **目標平台** | PC（滑鼠主導） |
| **回合模型** | 凍結→規劃→確認→播放（1x 速）→再凍結 |
| **移動** | 從 Player 向外拖曳滑鼠 → 放開 → 彈射；虛線預覽含玩家體積，自動判定沿地面/飛起 |

## 🕐 10. 回合制核心機制（Ronin 式）[DRAFT]

> **設計決策日期：2026-06-22**
> **方向：** Ronin 式純回合制 — 凍結→規劃→確認→動態播放→再凍結

### 10.1 回合流程模型（2026-06-22 最終確認）[DRAFT]

> **核心概念**：物理世界永遠是真實物理（重力、動量、碰撞），「回合制」是疊在上面的輸入調度層。每 0.3 秒物理推進一次，推進後凍結，玩家規劃下一動作。

```
[凍結] 觀察當前物理狀態 → 規劃一個行動 → 確認
[0.3s] 物理模擬推進 0.3 秒（Engine.time_scale = 1）
[凍結] 物理暫停（Engine.time_scale = 0）→ 觀察結果 → 規劃下一個
```

- **回合固定時長**：**0.3 秒**（不隨行動種類改變）
- **玩家在空中時也凍結**：彈弓弧線未落地 → 0.3 秒到 → 凍結在半空 → 下回合繼續飛（或選擇射繩針改向）
- **不等待動作完成**：針也是相同邏輯，飛行中每 0.3 秒凍結一次推進
- **時間實作**：凍結 = `Engine.time_scale = 0`；播放 = `Engine.time_scale = 1`

### 10.2 行動系統 — 情境感知輸入（2026-06-22 確認）[DRAFT]

**輸入方式由滑鼠行為自動決定，無需切換模式：**

| 滑鼠行為 | 預覽顯示 | 確認後效果 |
|---------|---------|----------|
| **從 Player 身上向外拖曳** | 玩家飛行弧線（含地面判定，自動決定滑行/飛起） | 彈弓：玩家被射出 |
| **左鍵點擊畫面任意位置** | 攻擊針虛線彈道 + 本回合到達點 | 攻擊針飛向目標 |
| **右鍵點擊畫面任意位置** | 繩針虛線彈道 + 本回合到達點 | 繩針飛向目標 |

**其他行動：**

| 行動 | 觸發方式 | 備註 |
|------|---------|------|
| **盪繩（獨立）** | 已有繩索錨點時，右鍵確認 | 以錨點擺盪一段弧度後落地 |
| **繩索改向（飛行中）** | 玩家飛行中途凍結時，右鍵點畫面 | 繩針飛向新目標，玩家弧線改向 |
| **斷開繩索** | 點繩針旁的「斷開」UI 按鈕 | 免費動作，不耗回合 |
| **回收針** | 靠近後點針旁的「回收」UI 按鈕 | 免費動作，不耗回合 |

**播放速度：正常速度（1x）。無「等待」回合，每回合必須選擇一個動作。**

### 10.3 針的飛行時間與多回合飛行（2026-06-22 確認）[DRAFT]

**核心機制：針可以跨多個回合持續飛行。**

- 針飛行速度：**2400 px/s**
- 每回合固定時長：**0.3 秒**
- **每回合針飛行距離：2400 × 0.3 = 720px**（約 56% 的 1280px 畫面寬）
- 若目標在 720px 內 → 本回合命中
- 若目標距離超過 720px → 針在 0.3s 後停在 720px 處凍結，下回合繼續飛 720px

**規劃時 UI：**
- 虛線顯示「本回合結束時針會在的位置」（不是最終落點，而是本回合到達點）
- 若目標在本回合可到達範圍內 → 虛線終點顯示命中預覽
- 若目標在本回合範圍外 → 虛線延伸到本回合可達點，提示需要多回合

**多回合飛行的戰術意義：**
- 玩家射出遠距針後，下一回合可以繼續其他動作（移動、射另一根針）
- 針和玩家的行動同步推進，形成「佈局交響」
- 不強制等待針落地才能行動（這是 A 方案的真正意義）

### 10.4 多針佈局策略 [DRAFT]

跨多個回合在場景中佈置針，然後組合使用：

```
回合 1：攻擊針 → 射入牆壁 A（針留在原地）
回合 2：攻擊針 → 射入牆壁 B（針留在原地）
回合 3：繩針 → 連到牆壁 A 的針，開始盪繩
回合 4：盪繩擺至高點，釋放 → 飛越場景
```

- 最多 5 根針同時在場景中
- 已插入場景的針視為「永久錨點候選」，直到被回收或擊退

### 10.5 行動範圍與 UI 顯示（2026-06-22 確認）[DRAFT]

**針的範圍（攻擊針和繩針共用）：**

- 虛線圓 = 本回合飛行距離的邊界

- 游標在虛線圓外 = 需多回合飛行，顯示本回合到達點（虛線延伸到邊界）

**彈弓的範圍：**
