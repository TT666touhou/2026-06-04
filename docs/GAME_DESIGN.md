## 🎮 1. 遊戲核心概念

### 1.1 核心定位 [DRAFT]
> 一款 **2D 橫向即時動作遊戲**，單人體驗。（GAP-073：原回合制設計已移除）
> 玩家以 WASD 即時移動，**Space 進入子彈時間**（time_scale=0.15）進行精準瞄準與多針佈局。
> 玩家持有**固定數量的鋼針**，在高速移動與子彈時間之間切換，組合針的位置與繩索動作。
> **即時走位 + 子彈時間規劃**是取勝核心。
> 靈感融合：**Max Payne**（子彈時間）× **Dimension W**（鋼針投擲精準戰鬥）× **Titan Souls**（一擊致命 + 武器回收）× **Ronin**（爬牆/天花板 + 繩索擺盪）

**設計靈感來源：**
- **Dimension W（漫畫）** — 主角 Kyouma 的戰鬥風格：非科技、純機械、近距投擲、精準打點
- **Titan Souls** — 一擊致命、武器（針）必須回收、Boss 弱點精確打擊

### 1.2 遊戲類型 [DRAFT]
- **主類型**：2D 橫向即時動作（Real-time Action Platformer with Bullet Time）
- **次類型**：資源管理動作（Resource Management Action）
- **人數**：單人（已確認，MMP-003 解除）
- **維度**：2D（橫向捲軸）

### 1.3 核心設計支柱 [DRAFT]
1. **資源即工具**：鋼針是移動工具也是武器，永遠有限，迫使玩家思考
2. **多功能單一機制**：同一根針根據使用方式產生完全不同的效果
3. **戰術精確**：在慢動作中從容瞄準——挑戰來自「思考正確方案」而非「反應速度」
4. **回收循環**：使用後必須回收針（或找到補充），製造節奏張力
5. **多針交響**：多根針同時在空中產生複合效果，佈局能力決定深度上限

## 🪡 2. 鋼針系統（核心機制）

### 2.1 針的基本屬性 [DRAFT]
- **持有數量**：5 根（固定上限，2026-06-20 GAP-039 由 3 提高）
- **物理屬性**：遵循彈道物理，可受地形彈射（TBD 細節）
- **回收方式**：靠近後點 UI 按鈕（詳見 §2.4；F 鍵已移除）
- **補充方式**：TBD（關卡道具？Boss 掉落？固定補充點？）

### 2.2 針的使用模式（GAP-073 即時制，2026-06-25 確認）[DRAFT]

**即時制：左/右鍵隨時可射，無回合限制。子彈時間（Space 按住）期間更從容瞄準。**

| 用途 | 輸入 | 行為 | 狀態 |
|------|------|------|------|
| **攻擊針** | **左鍵**點擊 | 發射飛行針（1600px/s），最遠 480px；命中後插入地形/敵人，留在場景中可回收（GAP-074）。子彈時間下以 0.15x 速度飛行 | [DRAFT] |
| **繩針（鉤抓/盪繩）** | **右鍵**點擊 | 發射飛行針（1600px/s），最遠 480px；命中建立 WireConstraint 錨點，繩擺即時生效；未命中靜默失敗（GAP-074）。子彈時間下以 0.15x 速度飛行 | [DRAFT] |
| ~~**鋼索平台**~~ | 已移除（GAP-041） | — | 已移除 |
| **拉動物體** | TBD | MVP 排除，後期實作 | [DRAFT] |
| **地形彈射** | 針飛行命中特定地形後 | 自動彈射到新方向（反射角），繼續飛行 | [DRAFT] 保留 |

> **射出機制（即時制，GAP-073）**：
> - 左鍵點擊 = 即時 raycast 攻擊針（480px），命中插入；可在子彈時間內精準瞄準
> - 右鍵點擊 = 即時 raycast 繩針（480px），命中建立錨點，玩家繩擺物理立即生效
> - 攻擊針與繩針回收：靠近後出現選項，點擊回收（不消耗行動）（設計待實作）
> - 可同時有攻擊針和繩針在場景中（無同一時刻限制）

### 2.3 細線（Wire）系統 [DRAFT]
**已確認（GAP-041 大幅簡化 + GAP-053 全面重寫）：**
- **只保留盪繩抓鉤**；平台機制、S 鍵穿透、Q/E/F 全部移除（GAP-041）。
- **視覺改為直線 Line2D**（GAP-053 移除 Verlet 12 段繩，改為 2 點直線）。

**帶線針物理 — 即時勾爪 + 手動收縮 + 自然鐘擺（GAP-053/064 重寫）[DRAFT]**

- **右鍵發射飛行針**（GAP-074）：右鍵點擊 → 發射 NeedleProjectile（1600px/s）飛向滑鼠方向，最遠 480px；命中牆面 → 即時建立 WireConstraint 錨點，繩擺物理立即生效；未命中 → 針消失，靜默失敗。子彈時間（time_scale=0.15）下針以 0.15x 速度飛行，演出明顯緩慢。
- **無自動收繩**（GAP-064）：繩長不會自動縮短，固定在命中瞬間距離。
- **手動收縮**（GAP-065/066）：錨點旁顯示「⬆ 收縮」UI 按鈕；點擊 → 在 0.3s 內平滑拉到錨點（rope_min_length=24px）；到達後自動釋放繩索；Player 若接觸牆/天花板則自動黏住。
- **Pre/Post 約束分離**（GAP-053 核心）：
  - `_apply_wire_pre`：`move_and_slide()` 前消除朝外徑向速度 + `snap_factor = 0.35` 彈力反饋
  - `_apply_wire_post`：`move_and_slide()` 後直接以 `global_position` 硬夾位置到繩長圓弧
  - **不注入任何速度脈衝**（消除速度分段問題）
- **WASD 移動 → 自動斷線**（GAP-073/064）：玩家 WASD 離牆或觸牆黏附時繩索自動釋放；攻擊針不斷線。
- **直線繩視覺**：2 點 Line2D（玩家→錨點），張力比例決定顏色與寬度（GAP-053）。
- **初始繩長**：命中瞬間精確距離（`wire_slack=0`）。
- **可調參數**：`rope_min_length=24`、`rope_snap_factor=0.35`。

### 2.3a 表面黏附（壁虎模式）（GAP-066/070）[DRAFT]

> **Player 碰到牆壁或天花板後自動黏住，重力暫停，直到採取主動行動。Ronin 機制：盪繩中碰牆同樣黏附，繩索自動釋放。**

- **觸發條件（GAP-070 Ronin 機制）**：即時移動中，`is_on_wall()` 或 `is_on_ceiling()` 為 true → 自動切換黏附狀態。**無論是否有繩索連接均觸發**；若繩索在線，附著前先釋放繩索（`_release_grapple()`）。
- **黏附效果**：速度歸零、重力取消；Player 固定在表面不滑落（壁虎物理）
- **平台邊緣掛住**：靠近平台側邊角時（雙向水平 raycast 偵測）→ 自動 snap 使手部位置對齊邊角，進入黏附狀態。**邊緣掛住僅在無繩索時觸發**（盪繩中不做邊緣偵測）。
- **解除黏附（主動）**：A/D 推離牆（按住遠離牆的方向）→ 解除；S 離開天花板 → 解除；解除後有 4 幀寬限期防止重黏（GAP-075）
- **解除黏附（被動）**：每幀逐 raycast 驗證表面是否還在。玩家 crawl 到天花板邊緣 / 爬牆超過牆頂 → 表面消失 → 立即自動釋放，允許重新黏附新表面（GAP-076）
- **繩索 + 黏附**：可在黏附狀態下發射勾爪繩索，點擊收縮時解除黏附並飛向錨點

### 2.4 鋼針回收規則（GAP-073 即時制版，2026-06-25）[DRAFT]

## 🏃 3. 移動系統（GAP-073：改為即時制）

### 3.1 WASD 即時移動（GAP-073）[DRAFT]

> 取代彈弓（Slingshot）移動。玩家直接以 WASD 即時控制角色。

**輸入：**
- **A/D** = 左右移動（walk_speed = 220 px/s）
- **W** = 跳躍（jump_force = 550 px/s，僅在地面時有效）
- **S** = 預留（目前無功能）
- **Space** = 進入**子彈時間**（time_scale = 0.15，保持按住）；放開恢復正常速度

**子彈時間（Bullet Time）：**
- 按住 Space → 全場物理與敵人減速到 15% 速度
- 玩家可在此期間精準瞄準、射針、連接繩索
- 放開 Space → 立即恢復正常速度
- aim preview（虛線預覽）只在子彈時間顯示

### 3.2 繩索擺盪（即時，GAP-073）[DRAFT]

> 繩索在即時制下持續生效，物理引擎計算鐘擺。

- **右鍵** → 即時 raycast（480px）勾住目標面 → 玩家開始擺盪（物理持續）
- 擺盪途中可以正常移動（A/D 不影響擺盪方向，重力主導）
- 繩索手動收縮（收縮按鈕）→ 0.25s 動畫縮至最短後自動釋放並黏附
- 碰到牆面/天花板 → 自動黏附（Ronin 機制，繩索釋放，見 §2.3a）

### 3.3 爬牆/爬天花板（GAP-066/070）[DRAFT]

- 碰到牆壁或天花板 → 自動黏附，重力取消
- **牆壁上**：W/S 沿牆上下爬行（wall_climb_speed = 140 px/s）；按遠離牆方向鍵 → 推離
- **天花板上**：A/D 沿天花板爬行；S → 鬆手下落
- **平台邊緣**：自動 snap 掛住邊角

### 3.4 移除的移動機制（GAP-073）
- ~~彈弓（Slingshot）拖曳發射~~：已移除，改用 WASD 移動
- ~~Space 跳躍~~：Space 改為子彈時間觸發；跳躍改為 W 鍵
- ~~回合制凍結規劃期間的 WASD 鏡頭平移~~：WASD 現在直接移動玩家

### 5.3 訓練木人（Training Dummy）[DRAFT]

> **用途**：機制驗證用的 MVP 敵人替身，用於測試針的命中、拉動與物理手感。

**基本規格：**

| 屬性 | 值 |
|------|-----|
| 節點類型 | `CharacterBody2D` |
| 佔位尺寸 | **32 × 64 px**（棕橘色 ColorRect，與 Player 等高） |
| 質量行為 | 有重力（落地後靜止；被拉動時移動） |
| 碰撞層 | Layer 1（與牆/地板/玩家一致） |
| HP | 無（純物理測試用） |

**針的互動行為：**

| 針種類 | 命中後效果 |
|--------|-----------|
| 攻擊針（左鍵） | **插入** + **一次性擊退衝量**（`knockback_force = 500 px/s`，方向：遠離玩家；地面上額外給一個小跳起） |
| 帶線針（右鍵） | **插入**（不產生拉動；帶線針保持盪繩錨點語義） |

**擊退邏輯（2026-06-21 實作）：**
- `on_needle_embedded(n_type)` — 由 NeedleManager 呼叫；攻擊針命中時立即計算擊退方向（dummy → player 反方向），疊加 `velocity += dir × knockback_force`；在地面時額外 `velocity.y -= knockback_force × 0.35`（小跳）
- 地面摩擦：`ground_friction = 800 px/s²`，使木人自然減速停止
- 無持續效果，無需 `on_needle_removed`

**場景位置（MVP_Test.tscn）：** `Vector2(900, 704)` — 地面上、玩家右側，需要用鉤爪接近。

## ⚡ 10. 即時制 + 子彈時間核心機制（GAP-073，2026-06-25）[DRAFT]

> **設計決策日期：2026-06-25**
> **方向：** 移除回合制。改為全即時 WASD 動作，Space 按住進入子彈時間（time_scale=0.15）。

### 10.1 時間模型

```
[即時] WASD 移動 + 左/右鍵射針 → 全程 time_scale=1.0
[Space 按住] 子彈時間 → time_scale=0.15（所有物理減慢到 15%）
[Space 放開] → 回到 time_scale=1.0
```

- **子彈時間效果**：玩家、敵人、物理一起慢下來（整個 Engine.time_scale 改變）
- **戰術用途**：在子彈時間內精準瞄準、佈置多針、觀察繩索弧線
- **子彈時間非無限**：[GDD TODO] 耐力/能量條設計待定
- **舊 TurnManager API**：`is_frozen()` = 子彈時間中；`commit()` = no-op（kept for compile）

### 10.2 輸入系統（即時制）

| 輸入 | 效果 |
|------|------|
| **A / D** | 水平移動（220 px/s） |
| **W** | 跳躍（地面時） |
| **S** | 保留（未分配功能） |
| **Space 按住** | 子彈時間（time_scale=0.15） |
| **左鍵** | 射攻擊針（即時 raycast 480px） |
| **右鍵** | 射繩針（即時 raycast 480px，建立 WireConstraint） |
| **左鍵（UI 按鈕）** | 收縮繩索 / 回收針（靠近時顯示，待實作） |

### 10.3 多針佈局策略（即時制版）[DRAFT]

在即時移動中多針同時佈置，子彈時間協助精準規劃：

```
即時移動 → 右鍵勾牆盪繩 → Space 進子彈時間
子彈時間內：左鍵射攻擊針插敵人 A → 再左鍵射攻擊針插敵人 B
放開 Space → 回即時，繼續盪繩飛行
```

- 最多 5 根針同時在場景中
- 已插入場景的針視為永久錨點候選，直到被回收
