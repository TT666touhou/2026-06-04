## 一、角色比例分析

### 角色頭身比（從 wiki GIF 和 Steam 截圖量測）

| 項目 | 數值 | 觀察 |
|------|------|------|
| 頭身比 | 約 1:2.5 ~ 1:3 | 頭部偏大，SD（Super Deformed）偏向但不極端 |

### 角色姿態（從 Steam 截圖）
- **遊戲視角**：2D 橫向卷軸，斜側視角（3/4 side view）
- **待機姿態**：輕微呼吸動作，重心略前傾
- **移動姿態**：流暢奔跑循環，帽子/斗篷有飄動效果
- **攻擊動態**：Snappy、快速，有明顯的 windup 和 follow-through

## 三、輪廓風格（從插畫確認）

- [x] **選擇性描邊（Selective Outline）** ← DS 主要風格
- [ ] 全黑描邊（Single Color Black Outline）
- [ ] 無描邊（Lineless）

### 描邊細節（從高解析度插畫確認）
- 外輪廓：深棕色（`#3D1C07` ～ `#5A2A0A`）而非純黑色
- 頭髮邊緣：使用更深的頭髮色而非黑色描邊（選擇性描邊典型）
- 衣物折痕：深色同色系描邊（非黑色）
- 描邊寬度：1px（標準），衣物輪廓偶爾 2px 加強
- 特效 VFX：紅色/白色/藍色發光（Glow 效果用柔化邊緣圓形遮罩）

### 高解析度插畫分析（已收集 26 張）— 重要發現

> ⚠️ **DS 使用兩種截然不同的插畫風格：**

| 風格 | 描述 | 代表範例 |

## 三、色彩特徵

### 從官方立繪 GIF 取樣（Assassin / Fighter / Doppelganger）

| 部位 | 主色（近似 HEX） | 陰影色（近似 HEX） | 備注 |
|------|----------------|------------------|------|
| 膚色 | `#F2C9A0` | `#D4956A` / `#B56A40` | 2 層陰影，乾淨不過深 |
| 髮色（棕） | `#C8904A` | `#8B5E30` | 偏暖棕，飽和度中等 |
| 髮色（青） | `#8EC4B8` | `#5A9490` | Fighter 青色系 |
| 白色衣物 | `#F5F0E8` | `#C8C0B0` | 偏米白而非純白 |
| 藍色衣物 | `#5080C0` | `#3060A0` | 飽和中藍 |
| 深棕輪廓 | `#3D1C07` | — | 外輪廓用色 |
| 描邊色（暗紅） | `#6B1A1A` | — | 內部陰影邊界 |

### 色彩層次規律
```
每個面的色彩分層：
  1層：主色（亮面）
  2層：主色陰影（暗面）
  3層：深色描邊/邊緣
  ← DS 通常只有 2-3 層，不使用漸層，保持 flat 風格
```

### 遊戲內配色（從 Steam 截圖）
- 場景主調：深灰磚牆 `#2A2A35`、棕紅 `#5A3020`
- 火把暖光：橙黃 `#F09030`（局部打光，非全場景打光）
- VFX：電光藍 `#60C8FF`、衝擊白 `#FFFFFF`、火花黃 `#FFD020`

## 四、動畫特徵

| 特徵 | 描述 |
|------|------|
| 動作節奏 | **Snappy**，攻擊快速有力，動作過渡帧數少 |
| 待機動作 | 呼吸循環，約 4~8 幀，微小搖擺 |
| 攻擊動作 | Windup（1~2 幀）→ Strike（1 幀）→ Recovery（2~3 幀） |
| 陰影層數 | **2 層**（主色 + 陰影色），個別部位 3 層 |
| 斗篷/衣物 | 有流體動感，不嚴格跟隨身體（半物理模擬感） |
| 受擊效果 | 角色閃白（flash white），附近有 hit spark |

## 五、DS 特有視覺特徵（從影片確認）

```
✦ 視角：純 2D 橫向卷軸（Side-scrolling），角色面向正側面或 3/4 側面
✦ 像素密度：高，在 64×96 空間（推估）堆疊大量細節
✦ 背景場景配色：
    - 磚牆/石牆：深灰棕 #2A2530 ～ #3D2A20
    - 木質平台：暗棕 #4A3020
    - 地板陰影：幾乎純黑 #1A1015
    - 火把/光源：橙黃暖光 #F09030（局部，非全場景）
✦ 角色與場景對比：角色顏色飽和度高，背景去飽和，強烈反差
✦ VFX（從影片確認）：
    - 攻擊 Slash：白色弧線剪影，帶淡藍色/橙色光量（不同技能顏色不同）
    - 大技/技能：亮紅/紫/金 能量環 + 大量 spark（見 rQXH8wkCF8s 金環）
    - 傷害數字：黃色粗體像素字 + 暗色描邊（正傷害）、紅色（受傷）
    - 暴擊：數字更大，帶閃白效果
    - 受擊：角色全身閃白 1 幀
✦ UI（從影片確認）：
    - 血量：左上角紅心格子
    - 右上角：Boss HP 條（紅色長條）
    - 底部：搖桿（左）+ 行動按鈕（右）— 移動平台 UI
    - 金幣/資源：左側顯示
✦ 角色重量感：落地有輕微頓挫，攻擊有明顯動作弧度（windup + follow-through）
✦ 角色比例（從影片估算）：
    - 遊戲內角色高度約佔螢幕高度 10-15%（從 360px 高推估約 36-54px）
    - 頭身比 ≈ 1:2.5，頭部偏大
✦ 相鄰角色對比：同畫面可有多個角色，設計上有明顯的體型差異（Boss 更大）
```

## 六、PixelLab Prompt 建議

基於以上分析，生成我們的主角（哥德蘿莉女孩）的建議參數：

```
核心提示詞（修正版）：
"pixel art character sprite, cute anime girl, red-orange hair in twin tails,
white ruffled blouse with puff sleeves, black gothic pleated mini skirt,
silver buckles, black ribbon bow tie, thigh-high stockings,
selective outline pixel art style, dungeon slasher game style,
2-3 color shading per area, flat cel-shaded pixel art,
side view for horizontal scroller, isolated on solid neon blue background,
32x32 pixels, crisp pixel edges, high detail for small sprite size"

風格參考（最相近的官方角色）：
→ 最接近參考：Trixie_Devil (Soul Eater) — 白銀雙馬尾 + 黑暗系洋裝 + 像素風格
→ 次要參考：Inquisitor Nachtrichter — 哥德系黑色外套 + 白色內搭
→ 服裝元素：Assassin 系列 — 多層衣物，裙子，蝴蝶結

關鍵風格限定詞：
- "selective outline pixel art" (非 black outline)
- "dungeon slasher style" / "96Percent game style"  
- "flat cel shading, 2-3 shade levels"
- "side view, horizontal scroller game sprite"
- "isolated on neon blue background" (#00FFFF 方便去背)

禁止使用的詞：
- "realistic shading" / "gradient" → 會導致非像素藝術感
- "lineless" → 會失去 DS 的描邊特色
- "cute chibi" → 比例會過度 SD 化
```

## 八、注意事項

1. **配色警告**：DS 角色用色偏暗飽和（dungeon 配色），禁止使用過於鮮豔的顏色
2. **頭髮紅色**：用戶要求紅色頭髮，DS 風格中應為 `#CC3030` ~ `#E05040` 的中飽和紅，不要太暗也不要螢光
3. **透明背景**：生成後需用 neon blue (`#00FFFF`) 去背，或 PixelLab 直接設置透明背景
4. **描邊顏色**：描邊不要用純黑 `#000000`，應為深棕/深色系（DS 的 selective outline 特徵）
5. **動畫幀數**：待機 4-8 幀，跑步 6-8 幀，攻擊 6-10 幀（Godot AnimatedSprite2D 最佳）
